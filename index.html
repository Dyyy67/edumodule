ℹ️ About

EduModule AI v1.1

AI-powered educational module generator for Grade 1–6 teachers. Uses Google Gemini AI for content and exports formatted .docx files (8.5"×13", 0.5" margins).

Built with Firebase, Gemini AI, docx.js, Tesseract.js, PDF.js.

❓ How to Use EduModule AI

Step 1: Get a Gemini API Key

Go to aistudio.google.com → Get API Key → Create API Key. It's free! Paste it in Settings → API Keys.

Step 2: Fill Module Details

Select grade, subject, and enter module title, teacher, and school. These appear on the cover page.

Step 3: Upload Source Material

Upload book pages as images, PDF, or Word files — or paste text directly.

Step 4: Generate & Export

Click "Generate Module." Preview the result, then download as .docx (Folio long, narrow margins, ready to print).
