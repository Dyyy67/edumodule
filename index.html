ℹ️ About

EduModule AI v1.0

An AI-powered educational module generator for Grade 1–6 teachers. Uses Google Gemini AI for content generation and exports to properly formatted .docx files.

Built with: Firebase, Gemini AI, docx.js, Tesseract.js
Page size: 8.5" × 13" (Folio/Long)
Margins: Narrow (0.5")

❓ How to Use EduModule AI

Step 1: Get a Gemini API Key

Go to aistudio.google.com → Get API Key → Create API Key. It's free! Paste it in Settings → API Keys.

Step 2: Fill Module Details

Select the grade level, subject, and enter the module title, teacher name, and school. These appear on the cover page.

Step 3: Upload Source Material

Upload book pages as images, PDF, or Word files. Or paste the text directly. The AI uses this as the lesson content.

Step 4: Generate & Export

Click "Generate Module." Preview the result, then download as .docx (8.5"×13", narrow margins, ready to print).

⚠️ Firebase Setup Required

To enable login and data saving, replace the Firebase config in the script section with your own project credentials from console.firebase.google.com. Enable Email/Password and Google auth in Firebase console.
